== Description ==

Completely brand the admin dashboard and login screen through easy and straight forward controls. Do advanced customizations and branding through adding your own CSS and JS. Fits both regular users and developers. Features include:

* Add your own logotypes to the admin dashboard, admin menu, and the login screen.
* Completely alter the look of the login screen.
* Easily set your own color theme for both the buttons and the complete admin section.
* Set your own login URL for your admin section. www.example.com/login for example.
* Make the WordPress admin menu look more fancy easily.

**Instructions**

After you've installed and activated the plugin you'll find a new entry in your WordPress menu called "Admin Branding". Click this to find all the customization options available to you.

== Installation ==

This section describes how to install the plugin and get it working. Recommended way of installation is by searching for the plugin in the 'Plugins' menu in WordPress. You can also follow the instructions below:

1. Download the zip-file containing the plugin and extract it.
2. Upload the `admin-branding` directory to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Done!
