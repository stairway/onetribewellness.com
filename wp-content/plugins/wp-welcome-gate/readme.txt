These are few uses of this plugin.

*	Acquire more subscribers to your email list
*	Get more traffic to your landing pages thus increasing conversions
*	Deliver a message to new visitors to your website
*	Introduce a new product or service to your audience

== Installation ==

Install Easy Facebook Login just as you would any other WP Plugin:

1.  [Download : wp-welcome-gate] from WordPress.org.

2.  Unzip the .zip file.

3.  Upload the Plugin folder (wp-welcome-gate/) to the wp-content/plugins folder.

4. Go to [Plugins Admin Panel](http://codex.wordpress.org/Administration_Panels#Plugins "Plugins Admin Panel") and find the newly uploaded Plugin, "Easy Admin CSS" in the list.

5. Click Activate Plugin to activate it.
